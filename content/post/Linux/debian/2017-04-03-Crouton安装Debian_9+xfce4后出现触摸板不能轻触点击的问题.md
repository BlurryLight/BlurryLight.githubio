这个问题蛮少见的，因为完整安装Debian的话应该是会带这个驱动，但是利用Crouton安装出来的Debian没有，所以导致我的Chromebook在Linux下触摸板工作的不太正常。
```
sudo apt install xserver-xorg-input-synaptics
```
然后在xfce4的设置—鼠标与触摸板中会多出触摸板的选项。
